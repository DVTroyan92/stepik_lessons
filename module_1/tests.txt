- Регистрация пользователя 1.1
-- Проверка функционала регистрации пользователя 1.1.1
Предусловия:
Имеется электронная почта, по которой ранее не производилась регистрация
Имеется электронная почта, по которой ранее была произведена регистрация
Шаги:
- Шаг 1, регистрация с помощью нового адреса электронной почты, поля "Пароль" и "Повторите пароль" совпадают, регистрация прошла успешно
- Шаг 2, регистрация с помощью нового адреса электронной почты, поля "Пароль" и "Повторите пароль" не совпадают, возникло сообщение об ошибке
- Шаг 3, регистрация с помощью зарегистрированного адреса электронной почты, поля "Пароль" и "Повторите пароль" совпадают, возникло сообщение об ошибке
- Шаг 4, регистрация с помощью зарегистрированного адреса электронной почты, поля "Пароль" и "Повторите пароль" не совпадают, возникло сообщение об ошибке

- Авторизация пользователя 1.2
-- Проверка функционала авторизации пользователя 1.2.1
Предусловия:
Создана учетная запись, к которой имеются пароль и адрес электронной почты 
Шаги:
- Шаг 1, авторизация с помощью зарегистрированного адреса электронной почты и корректным паролем, авторизация прошла успешно
- Шаг 2, авторизация с помощью зарегистрированного адреса электронной почты и некорректным паролем, возникло сообщение об ошибке
- Шаг 3, авторизация с помощью незарегистрированного адреса электронной почты и паролем от зарегистрированной учетной записи, возникло сообщение об ошибке
- Шаг 4, авторизация с помощью незарегистрированного адреса электронной почты и произвольно введенным паролем, возникло сообщение об ошибке

- Поиск товара 2.1
-- Проверка функционала поиска товара через поисковую строку 2.1.1
Предусловия:
В магазине имеются товары, как с уникальным названием, так и похожие им друг на друга
Шаги:
- Шаг 1, поиск через поисковую строку товара по полному наименованию, имеющегося в ассортименте, поиск выполнен успешно, товар отображается в результате поиска 
- Шаг 2, поиск через поисковую строку по названию по неполному наименованию, общему для нескольких товаров, имеющихся в ассортименте, поиск выполнен успешно, несколько товаров, название которых попадает под введенное в поле поиска значение, отображаются в результате поиска
- Шаг 3, поиск через поисковую строку товара, не имеющегося в ассортименте, возникло сообщение о том, что ничего не найдено
- Добавление товара в корзину 2.2
-- Проверка функционала добавления товара в корзину 2.2.1
Предусловия:
В магазине имеются товары с пометками "На складе" и "Нет на складе"
Шаги:
- Шаг 1, добавить в корзину товар, имеющий пометку "На складе", появилось уведомление о добавлении товара в корзину, в самой корзине он также присутствует
- Шаг 2, добавить в корзину товар, имеющий пометку "Нет на складе", кнопка "Добавить в корзину" не активна, при нажатии на нее ничего не происходит, в самой корзине товар также отсутствует
